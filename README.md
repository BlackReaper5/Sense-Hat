# Sense-Hat
This is the Sense Hat which has 6 sensors ->
Gyroscope
Accelerometer
Magnetometer
Temperature
Barometric pressure
Humidity
For this project I only use the Temperature, Barometric pressure and Humidity sensors.
The fetched data is being send to ThingSpeak and fetched by the WeatherApp ->
https://github.com/BlackReaper5/WeatherApp
